Toolbar

Horizontal collection of inputs and filters

Basic structure

Toolbar is a wrapper to organize multiple equal-height cells into perfectly aligned row. It takes full row of horizontal space.

    Options

    Additional modifiers can be assigned to main .toolbar element

    Cells Autowidth

    Add class .toolbar-flex to make cells fill entire space

    You can specify your own desired width for every cell (in pixels or percents) or use predefined .w50 and .w100 classes for 50%/100%.

    <div class="toolbar tolbar-flex">
    <div class="toolbar__item w50">Component 1</div>
    <div class="toolbar__item w50">Component 2</div>
</div>
