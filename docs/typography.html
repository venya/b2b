Typography

Fonts and Text styling

Fonts

Recommended font family for Google Material Design application is Roboto. This is defined in _variables.scss and can be easily changed.

Don't forget to include font files in web-application static folder (for operating systems other than Androi 4.0+)

Text size

Base font size configured in _variables.scss and used globally for generic texts. Components may define its own font sized, preferably based on thia base font size.

You can explicitly change font size via following classes:

.small

Small text size

.text-uppercase

Make any text uppercase

.big

Bigger text size

.xl

Very big text size

Colors

You can override any element style assigning one of the following classes:

Text Color

.muted For grayed (muted town) color. Used for less important or disabled elements.

This is muted text

.text-success Green color for text with positive meaning.

Success text

.text-danger Red color when come attention needed.

Danger text

.text-primary Blue color matching primary buttons.

Alignment

This is most helpful utility classes, used for alignment of individual elements

.pull-left Set float:left for some element

.pull-right Set float:left for some element

Horizontal alignent modes: often used in table cells
