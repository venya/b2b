Buttons

Building different page structure

Basics

Different implementations. Almost any element including link, button, input, div or span can be turned to button by adding .btn class. They will looks the same way:

Link Button Span
Div
<a class="btn">Link Button</a>
<span class="btn">Span</span>
<div class="btn">Div</div>
<button class="btn">Button</button>
<input type="button" class="btn btn-default" value="Input Button">

Color Options

When only .btn class specified we're not setting any color to button, so you can base your own customized buttons on this skeleton.

Use default color (.btn-default) to make button color persistent and not browser-dependable:

Styling via additional .btn-default, .btn-primary, .btn-secondary, .btn-success, .btn-white and .btn-link (for trasparent text-like appearence)

Additional Styling

Styling options: .btn-rounded, with .icon inside, .btn.icon

Icons can be placed inside button (except <input type=button/submit/cancel> where no child elements allowed)
