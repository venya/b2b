Tables

Used to display data in tabular format.

Basic

Use regular HTML <table> with basic class .table.

Put table header in <thead> and data rows inside <tbody> (they have different styling).

<div class="alert">
    Simple alert message
</div>

Additional Table styling

Hover

.table-hover modified can be added to table element to show hover mouse effects on rows. This is used when user can click entire table row for executing some action. That is why pointer cursor diplayed and they have :active effect.

Please note, hover effects applied only to <tbody>

Example:

IDTitle
1First line
2Second row
3Last third row

Striped

Interleved background color for even/odd rows. Add .table-striped class to table. This effect can be combined with others, eg hover.

IDTitle
1First (Odd) line
2Even row
3One more Odd row
4Last

Bigger size

Assign .table-lg to increase text size and table paddings for more spacier data presentation.

Individual cells

Alignment

Horizontal alignment defaults to left. You can change whit via additional classed assigned to TD or TH: .text-left, .text-center, .text-right

Vertical alignment defaults to middle. You can change whit via additional classed assigned to TD or TH: .valign-top, .valign-middle, .valign-bottom

Narrow column

Sometimes there is need to limit column width to minimal possible (eg: when it contain only one icon or checkbox). You can add .table__cell-minimize

First/Last cell spacing

You can selectively increase left padding for cells in first column, and right margin in last column. Use corresponding cell classes .table__cell-first or .table__cell-last

This is useful when table inserted directly into page or modal dialog spawning from edge to edge (without parent with horizontal paddings).

Grids

Used for custom layouts in 2, 3 or 4 columns, that can have multiple rows when number of element more than fits in one row. Useful cor .card layouts

They structured from container (base class .grid with column number specified .grid-3-col) with multiple cells (.grid__item) inside, that can have spacing or not.

Example: 3 columns, no spacing: .grid .grid-3-col
