Icons

High resolution Icons via Font-based glyphs

Icon font

Vector-based icons graphics implemented via custom web-font, which allows to reduce download footprint while maintaining higher possible quality on every resolution (inclung Mac/iOS Retina displays and HiDPI Android devices)

Web font loaded automatically inside _icons.scss and loaded locally from following files:

/static/fonts/icomoon.woff
/static/fonts/icomoon.ttf
/static/fonts/icomoon.eot
/static/fonts/icomoon.svg

Icomoon.io

Free online service IcoMoon.io used to generate this files from public icons gallery. Mostly used Google Material Design Icons.

Adding New Icons

  1. Goto IcoMoon.io application
  2. Import project saved in XML file in project TFS folder
  3. Add/Replace glyphs
  4. Set unique IDs for new icons
  5. Generate Font -> Download
  6. Replace static font files (woff, ttf, eot, svg)
  7. Insert list of glyphs definitions (looking like .icon-*:before) into _icons.scss
  8. Recompile SASS sources

How to use icons

Assign .icon class to any inline, block or inline-block element to apply icon font (along with some special syling)

Assign additional class, like .icon-pos to insert specific icon glyph from available library.

<i class="icon icon-pos"></i>

Icons displayed as inline-block elements with default size of 40×40 pixels.

Modifiers

.icon-sm can be used to make icon smaller (20×20 pixels):

.baseline to align icon inline with other text (vertical-align: middle)

Alternatively you can assign .icon-text instead of .icon to other elements without turning them into inline-blocks, just apply icon font and some rendering optimizations.
