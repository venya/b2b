Dropowns

Replacement for standard HTML <select>

Base element should have .dropdown class and have inside following components:

  • .dropdown__selected representing currently selected item and/or title. Can contain optional icon (.dropdown__selected-icon)
  • .dropdown-menu with available items
  • .dropdown-menu__mask (optional) background mask, that appears when dropdown is open
<div class="dropdown">
    <div class="dropdown__selected">
        <div class="icon dropdown__selected-icon"></div>
        <div class="dropdown__selected-label">Option:</div>
        <div class="dropdown__selected-value">With Icon</div>
    </div>
    <ul class="dropdown-menu">
        <li class="dropdown-menu__item">Option 1</li>
        <li class="dropdown-menu__item">Option 2</li>
    </ul>
    <div class="dropdown-menu__mask"></div>
</div>
