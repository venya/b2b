Inputs

Form elements

Basic usage

Add basic .input class to any <input>, <textarea> or <select> element.

Options

Size

5 sizes available: normal, small .input-sm, extra-small .input-xs, large .input-lg and extra large .input-xl

Colors

Additional styling classes:
Unstyled
.input-default Default light gray
.input-white White background
.input-silver Silver gray
.input-transparent Transparent

Highlight focus

Add following class to have blue outline when focus in input .input-focus-highlight

Full width

You can add .w100 class to make input full-with
